# Лабиринт сокровищ

Текстовая adventure-игра в консоли. Игрок бродит по комнатам древнего лабиринта,
собирает предметы, решает загадки, активирует ловушки и пытается добраться до
комнаты сокровищ.

## Установка

```bash
make install
# или
poetry install


## Запуск игры

make project
# или
poetry run project
